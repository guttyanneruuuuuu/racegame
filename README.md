# 🏁 GyroRush - スマホジャイロ3Dレースゲーム

![GyroRush](https://img.shields.io/badge/Game-Racing-red) ![Three.js](https://img.shields.io/badge/Three.js-r160-blue) ![PeerJS](https://img.shields.io/badge/PeerJS-WebRTC-green)

**スマホを傾けてハンドル操作！** 友達6人までで対戦できる3Dレースゲームです。
マリオカート風のアイテムシステムで他のプレイヤーを妨害したり、ブーストで一気に逆転！

## 🎮 遊び方

### 📱 スマホ
1. **横向き** にスマホを持つ
2. **スマホを左右に傾ける** → ハンドル操作
3. **画面右の ACCEL** → アクセル
4. **画面左の BRAKE** → ブレーキ / バック
5. **画面右の ★USE** → 取得したアイテムを使用

### 🖥 PC (キーボード)
- **↑ / W** : アクセル
- **↓ / S** : ブレーキ / バック
- **← / A** : 左
- **→ / D** : 右
- **Space / X** : アイテム使用

### 🎯 マウスでもハンドル操作
- 画面中央を左右にドラッグするとハンドルを切れます（バックアップ操作）。

## ✨ 主な機能

### 🌐 マルチプレイヤー (最大6人)
- 部屋を作る → 6文字のコードを友達にシェア
- 友達はコードを入力して入室
- WebRTC (PeerJS) による P2P 通信。サーバーレスで GitHub Pages だけで完結！

### 🎁 アイテム (5種類)
| アイテム | 効果 |
| :--- | :--- |
| ⚡ BOOST | 2.5秒間 加速！ |
| 🚀 ROCKET | 前方の敵に追尾ミサイル発射 (爆発エフェクト付き) |
| 🍌 BANANA | 後ろにバナナ設置。踏むとスピン |
| ⛈ LIGHTNING | 全プレイヤーを一時操作不能に |
| 🛡 SHIELD | 5秒間 無敵 |

順位が低いほどレア度の高いアイテムが出やすくなっています（後方プレイヤー優遇）。

### 🤖 ソロモード（ひとりで練習）
- AI 5体（個別スキル + ライン取り）と対戦できるオフラインモード

### 🎬 3D グラフィックス & 演出
- Three.js による 3D レンダリング
- 滑らかなCatmull-Romコース（コース幅広め＋壁あり）
- 縁石（赤白チェック）/ 立体的なバリア / スタートゲート
- 青空＋雲のスカイドーム / 芝生のテクスチャ / 観客スタンド / 旗 / 木々
- ブースト時の炎エフェクト、ドリフトスモーク、ロケット爆発
- 速度に応じたカメラ追従 + FOV変化、衝突時のカメラシェイク

### 🖥 HUD
- スピードメーター (km/h)
- ラップカウンタ (1/3, 2/3, 3/3)
- 現在順位 (1/6 など)
- タイマー & ベストラップ
- ミニマップ（コース全体 + 全車位置）
- リアルタイム順位リスト
- ステアインジケーター

### 🎛 操作の調整
- ジャイロ「再設定」ボタン（タップで再キャリブレーション、長押しで感度スライダー表示）
- 感度スライダー（最大角度 8°〜35°）で自分の好みに
- ゲーム中UIは右上に再設定・感度・ステア表示を集約して視界を確保
- ジャイロ舵方向は最新調整版（従来と逆向き）に更新

## 🎥 カメラ
- 車のすぐ後ろ・低い位置からの三人称視点で臨場感
- 狭めの基本FOV（58°）+ スピードでFOV広がる演出
- ブースト時はさらに広がってスピード感倍増

## 🛡 壁とコース
- コース幅広め＋立体的な壁で**飛び出しにくい設計**
- 壁にぶつかると速度減衰＋カメラ揺れ＋車体が一瞬赤くフラッシュ
- 路面外は重い摩擦で減速

## 🎯 ゲームのコツ

- **インを攻める** ことでショートカット（壁すれすれを攻めろ！）
- **コース外は減速** するので注意！
- **後ろの順位だとレアアイテムが出やすい** ので逆転チャンス
- **シールド中はバナナや雷を防げる**
- **ロケットは前方の敵を自動追尾** するので、後ろから狙え
- **ジャイロは横向きで使う** とハンドル操作に対応
