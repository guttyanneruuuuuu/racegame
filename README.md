# 🏁 GyroRush - スマホジャイロ3Dレースゲーム

![GyroRush](https://img.shields.io/badge/Game-Racing-red) ![Three.js](https://img.shields.io/badge/Three.js-r160-blue) ![PeerJS](https://img.shields.io/badge/PeerJS-WebRTC-green)

**スマホを傾けてハンドル操作！** 友達6人までで対戦できる3Dレースゲームです。
マリオカート風のアイテムシステムで他のプレイヤーを妨害したり、ブーストで一気に逆転！

## 🎮 遊び方

### 📱 スマホ
1. **横向き** にスマホを持つ
2. **スマホを左右に傾ける** → ハンドル操作
3. **画面右の ACCEL** → アクセル
4. **画面左の BRAKE** → ブレーキ / バック
5. **画面右の ★USE** → 取得したアイテムを使用

### 🖥 PC (キーボード)
- **↑ / W** : アクセル
- **↓ / S** : ブレーキ / バック
- **← / A** : 左
- **→ / D** : 右
- **Space / X** : アイテム使用

### 🎯 マウスでもハンドル操作
- 画面中央を左右にドラッグするとハンドルを切れます（バックアップ操作）。

### 🌀 ドリフト + ミニターボ システム ⭐ NEW
- ハンドル切りながらブレーキで **ドリフト開始**
- チャージが溜まると 3 段階の火花色:
  - **青火花** (Lv1) → 短いブースト
  - **黄火花** (Lv2) → 中ブースト
  - **紫火花** (Lv3) → 超ブースト！
- ブレーキ離した瞬間にミニターボ発動

### 🌪 スリップストリームターボ ⭐ NEW
- 前方のマシンの真後ろに入ると空気抵抗が下がって **加速ボーナス**
- 追走を維持するとゲージが溜まり、満タンで **SLIPSTREAM TURBO** 発動
- 直線での駆け引きや、抜き返しのタイミングがさらに熱く！

### 🔄 自動復帰 & 逆走警告
- 速度 1.5 m/s 未満が **2.5 秒続いたら自動リスポーン** (壁に挟まっても安心)
- 逆走が続くと画面中央に「⚠ 逆走中！」警告 → そのまま続行で**自動方向修正**に移行
- 自動方向修正中は「🛟 方向修正中…」表示 + 吊り上げ回転風の演出後に進行方向へ戻る
- 手動でも 🔄 ボタン / R キーでリスポーン可

### 🤖 ソロモード（ひとりで練習）
- AI 5体（個別スキル + ライン取り）と対戦できるオフラインモード

### 🎛 操作の調整
- 右上の ⟲ ボタンでジャイロ再キャリブレーション、⚙ ボタンで感度スライダー表示、🔊 ボタンでミュート切替
- 感度スライダー（最大角度 8°〜35°）で自分の好みに
- **反転チェックボックス**でジャイロ方向を反転（持ち方の好み対応）
- **20サンプル自動キャリブレーション**: レース開始時に持ち方を自動学習
- **低パスフィルタ**でジャイロのブレを吸収して滑らかなステアリング
- ゲーム中UIは画面端に集約して視界を確保（中央には何も置かない）
- 👀 ボタンで一時的に後ろを振り返り、🔄 ボタン / R キーで手動リスポーン

## 🎥 カメラ
- 車のすぐ後ろ・低い位置からのコックピット寄り視点
- 車が画面の約1/3を占める近接視界で臨場感MAX
- 狭めの基本FOV（52°）+ スピードでFOV広がる演出
- ブースト時はさらに広がってスピード感倍増
- ジャンプ時はカメラもふわっと持ち上がる

## 🌋 コース: VOLCANO CIRCUIT (火山サーキット)
真っ黒い火山岩アスファルト + 燃える赤紫の空 + 遠景の溶岩湖。
全く違うテーマ統一マップで、コースレイアウトも一新（交差なしの複雑な単一閉ループ）。

### ギミック / 障害物
- **⚡ ダッシュ盤 (溶岩オレンジ)**: 一気にブースト！12箇所配置（当たり判定半径 3.2）
- **💨 間欠泉ジャンプ台**: 火山岩リング+脈動する蒸気柱。4箇所、蒸気が伸縮するアニメ付き（半径 4.0、空中再発動防止）
- **🔥 溶岩プール (NEW)**: 触れるとスリップ + 速度45%カット + 赤い画面フラッシュ。3箇所
- **🪨 転がる溶岩岩 (NEW)**: コース幅を行ったり来たりする動く障害物。激突で吹き飛び。3個
- **🩸 溶岩割れ目ショートカット**: 内側を攻めると先回り（コース外ペナルティ免除）
- **💎 クリスタル / 岩柱**: 雰囲気装飾。共有マテリアル/ジオメトリで超軽量
- アイテムボックスはレインボー6色面+光るリング+**光柱ビーム**で視認性UP（半径 2.4）

### ⚡ 軽量化
- セグメント分割 16 → **12** (頂点約25%減)
- 全テクスチャ 128px → **64px**
- 地面サイズ 1200 → **800**、スカイドーム 600 → **450**
- 装飾は **共有ジオメトリ/マテリアル** で draw call 削減（岩柱/小石/クリスタル/岩石）
- アイテムボックスのリング/ビームジオメトリも全箱で共有
- 観客スタンド canvas (描画コスト高) を削除、岩柱に置換

## 🎯 ゲームのコツ

- **インを攻める** ことでショートカット（壁すれすれを攻めろ！）
- **草地ショートカット** を見つけて先回り！
- **コース外は減速** するので注意！（ショートカットゾーンは例外）
- **後ろの順位だとレアアイテムが出やすい** ので逆転チャンス
- **シールド中はバナナ・雷・インク・地雷も完全防御**
- **ロケットは前方の敵を自動追尾** するので、後ろから狙え
- **トリプルロケット**は上位3台を一網打尽
- **ドリフト中にチャージを溜めて紫火花**で爆発的加速
- **ゴースト中は他車をすり抜け**でショートカット
- **マグネット中はアイテムボックスが吸い寄せ**で連続取得
- **逆走警告 (⚠) は自動方向修正が入るので、慌てず姿勢を整えると復帰しやすい**
- **ジャイロは横向きで使う** とハンドル操作に対応
