# 🏁 GyroRush - スマホジャイロ3Dレースゲーム

![GyroRush](https://img.shields.io/badge/Game-Racing-orange) ![Three.js](https://img.shields.io/badge/Three.js-r160-blue) ![PeerJS](https://img.shields.io/badge/PeerJS-WebRTC-green)

スマホを傾けてハンドル操作！友達6人までで対戦できる3Dレースゲームです。マリオカート風のアイテムシステムで他のプレイヤーを妨害したり、ブーストで一気に逆転を狙え！

## 🎮 遊び方

### スマホ
1. **横向き** にスマホを持つ
2. **スマホを左右に傾ける** → ハンドル操作
3. **画面右の ACCEL** → アクセル
4. **画面左の BRAKE** → ブレーキ / バック
5. **画面中央の ★ アイテム** → 取得したアイテムを使用

### PC (キーボード)
- **↑ / W** : アクセル
- **↓ / S** : ブレーキ
- **← / A** : 左
- **→ / D** : 右
- **Space / X** : アイテム使用

## ✨ 主な機能

### マルチプレイヤー (最大6人)
- 部屋を作る → 6文字のコードを友達にシェア
- 友達はコードを入力して入室
- WebRTC (PeerJS) による P2P 通信。サーバーレスで GitHub Pages だけで完結！

### アイテム (5種類)
| アイテム | 効果 |
| :--- | :--- |
| ⚡ BOOST | 2.5秒間 加速！ |
| 🚀 ROCKET | 前方の敵に追尾ミサイル発射 |
| 🍌 BANANA | 後ろにバナナ設置。踏むとスピン |
| ⛈ LIGHTNING | 全プレイヤーを一時操作不能に |
| 🛡 SHIELD | 5秒間 無敵 |

順位が低いほどレア度の高いアイテムが出やすくなっています（後方プレイヤー優遇）。

### ソロモード（ひとりで練習）
- AI 5体と対戦できるオフラインモード

### 3D グラフィックス
- Three.js を使った 3D レンダリング
- 滑らかな曲がりのCatmull-Romコース
- スタートゲート、観客スタンド、木々、タイヤバリアなど装飾豊富

### HUD
- スピードメーター (km/h)
- ラップカウンタ (1/3, 2/3, 3/3)
- 現在順位 (1/6 など)
- タイマー
- ミニマップ
- リアルタイム順位リスト
- ステアインジケーター

## 🎯 ゲームのコツ

- **インを攻める** ことでショートカット
- **コース外は減速** するので注意！
- **後ろの順位だとレアアイテムが出やすい** ので逆転チャンス
- **シールド中はバナナや雷を防げる**
- **ロケットは前方の敵を自動追尾** するので、後ろから狙え
